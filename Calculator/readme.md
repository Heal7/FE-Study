# 计算器

## 功能：

+   点击“C”清空
+   可连续进行加、减、乘、除、取模二元运算（变量内不能有多个小数点，除数不为0）
+   特殊处理：

    + 依次输入变量-运算符-等号，输出为变量；
    + 无左侧变量时，默认为0；
    + 连续输入运算符时，用后一运算符替换前一运算符；
    + 连续输入等号-变量/小数点，重新开始一个表达式；连续输入等号-运算符，则继续计算。
